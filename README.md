# Diacomp
Diabetes Diary & Analytical System

<b>Features:</b>
  * Tracking patient data (BS measurements, meal, insulin dosage etc.)
  * Making sophisticated adaptive math model
  * Insulin dosage calculating
  * Shared extendable food/dish databases
  * Visual analyzing

<b>Platforms:<b/>
  * Android
  * Web (Java)
  * Windows (Delphi)
