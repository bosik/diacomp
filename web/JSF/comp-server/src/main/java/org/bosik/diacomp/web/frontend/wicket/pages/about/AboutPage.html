Что такое Компенсация?
Компенсация — это бесплатный электронный дневник, предназначенный для больных сахарным диабетом I типа.

Для чего она нужна?
Программа позволяет вести дневник, записывая в него данные о значениях сахара крови (СК), дозах инсулина и всей съеденной пище. Главные отличия Компенсации от других программ — возможность комплексного анализа полученной информации; автоматическое вычисление и адаптация коэффициентов, расчёт необходимой дозы инсулина; балансирование рациона.
